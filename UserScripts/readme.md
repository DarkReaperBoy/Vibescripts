# Installing Violentmonkey on Android

> *[Every journey's gotta start with a step](https://www.youtube.com/watch?v=YS-ABmWgi6M)*

### Step 1 — Download the Extension File
Download the [Violentmonkey `.xpi` file](https://addons.mozilla.org/firefox/downloads/file/4704557/violentmonkey-latest.xpi) using Chrome or your preferred download manager. Once downloaded, move it somewhere easily accessible on your device.

### Step 2 — Install a Firefox-Based Browser
Install [Fennec F-Droid](https://f-droid.org/packages/org.mozilla.fennec_fdroid/) or any other Firefox-based browser on your Android device.

### Step 3 — Unlock the Debug Menu
Open Fennec (or your chosen Firefox-based browser) and navigate to **About Fennec** in the settings. Once there, rapidly tap the **Fennec F-Droid** logo at the top several times until a debug menu option appears.

### Step 4 — Install the Extension
Return to the main settings page. You should now see an **Install Extension from File** option — tap it, then locate and select the `.xpi` file you downloaded in Step 1.

### Step 5 — Add Your Userscript
Find the userscript you'd like to use, copy its contents, and paste it into Violentmonkey's editor to get started.
